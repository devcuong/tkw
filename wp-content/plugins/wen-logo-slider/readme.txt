== Description ==

<h3>WEN Logo Slider</h3>
WEN Logo Slider by WEN Solutions, is a simple solution for carousel-based affiliation , in mind, but can be used and extended to a wide variety of purpose for responsive, beautiful display of slideshow. It comes with standard options that gives the admin control over the look and feel of the slider.

**Features**

* Fully Responsive slider
* Simple and easy to use
* External link for each slide
* Options for slide speed, transition delay, number of logos per slide
* Show/hide navigation arrow in slider
* Option to specify image size
* Uses WordPress Media Library to upload images
* Easy re-ordering of images ( drag-drop )
* Option for random ordering of images
* Multiple slideshows
* Powered by [Slick](http://kenwheeler.github.io/slick/)

**More Pro Features**

WEN Logo Slider Pro: [Detail](http://themepalace.com/downloads/wen-logo-slider-pro/) | [Demo](http://wensolutions.com/plugins/wen-logo-slider-pro/demo)

* Have two different settings one for specific slider and another settings for all sliders.
* Show Slider Title option for show/hide slider title.
* Title size option for title size of slider.
* Have Auto Play option to play slider in auto run mode or not.
* Pause on hover option for slider images to provide option for pause on hover.
* Center Mode option for creating a center mode of slider.
* Caption position option for changing caption position of slider.
* Random Order to specify whether order slider images in random order or not.
* Enable Mobile Option for specifying image resolution of images in slider for mobile device.
* Image Per Slide in option for managing image resolution number in mobile device.
* Image Size	option for managing size of image in sliders.
* Loop Slide option for managing looping of slider images.
* Mouse Dragging specify images in slider are mouse draggable or not.
* Slide Direction Right to Left option specifies direction for slider images.
* Lazy Load specify whether lazy load a slider or not.
* Pagination option for managing slider images pagination.
* Pagination Type option for managing slider images pagination type like numeric, simple, thumb.
* Navigation Arrow option for deciding whether to show navigation arrow or not.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for `wen-logo-slider`
3. Click `Install Now`
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `wen-logo-slider.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wen-logo-slider.zip`
2. Extract the `wen-logo-slider` directory to your computer
3. Upload the `wen-logo-slider` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
